ReCaptcha_v3

Основанный на токенах метод автоматического решения reCAPTCHA V3.

Полученный токен может быть отправлен на целевой веб-сайт внутри поля формы g-recaptcha-response или передан функции обратного вызова.

Спецификация для метода solve_captcha
Свойство Тип Значение Описание
captcha_type string recaptcha_v3 Поле для обозначения решаемого вида каптчи
service string RuCaptcha/CaptchaGuru/AntiCaptcha Сервис для решения каптчи
minScore string Обязательное значение score, примеры значений: 0.3 0.7 0.9
site_key string reCAPTCHA sitekey. Свойство data-sitekey для reCAPTCHA, которое можно найти внутри элемента div или внутри параметра k запросов к reCAPTHCHA API.
page_url string url Полный URL-адрес целевой веб-страницы, на которую загружается капча.
action string Значение параметра action. Значение устанавливается владельцем веб-сайта внутри свойства data-action элемента reCAPTCHA div или передается внутри объекта options при вызове метода execute, например grecaptcha.execute('websiteKey'{ action: 'MyAction' })

"service": "rucaptcha",

"site_key": "6LfB5_IbAAAAAMCtsjEHEHKqcB9iQocwwxTiihJu",
"page_url": "https://rucaptcha.com/demo/recaptcha-v3",
"action": "demo_action"
}

Turnstile

Основанный на токенах метод обхода капчи Cloudflare Turnstile. Поддерживаются два вида капчи Cloudflare Turnstile, как Standalone Captcha, так и Cloudflare Challenge page.

Standalone Captcha - виджет капчи размещен на странице веб-сайта, смотрите нашу демо страницу Turnstile


Чтобы обойти Standalone Captcha, вам просто нужно найти sitekey, получить токен из нашего API, а затем передать его вашей целевой странице в качестве значения элемента ввода с именем cf-turnstile-response или g-recaptcha-response для режима совместимости с reCAPTCHA. Также токен может быть обработан с помощью функции обратного вызова, переданной в вызов turnstile.render.

Спецификация для метода solve_captcha
Свойство Тип Значение Описание
captcha_type string recaptcha_v3 Поле для обозначения решаемого вида каптчи
service string RuCaptcha/CaptchaGuru/AntiCaptcha Сервис для решения каптчи
site_key string Turnstile sitekey. Значение sitekey можно найти внутри свойства data-sitekey элемента div Turnstile капчи
page_url string url Полный URL-адрес целевой веб-страницы, на которую загружается капча.

Пример запроса

Метод: solve_captcha

{
"captcha_type": "turnstile",
"service": "rucaptcha",
"site_key": "0x4AAAAAAAVrOwQWPlm3Bnr5",
"page_url": "https://rucaptcha.com/demo/cloudflare-turnstile"
